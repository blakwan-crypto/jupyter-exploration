# jupyter-exploration

## Lab Session: Introduction to GitHub and Jupyter Notebooks

This repository is part of the ITAI 1378 course lab session focused on exploring GitHub and Jupyter Notebooks. During this lab, we:

- Set up a GitHub account and created this repository
- Explored the GitHub interface, including repositories, issues, and pull requests
- Accessed Jupyter Notebooks using Google Colab (an online environment)
- Created a Jupyter Notebook with Markdown and Python code cells
- Ran code cells and observed outputs
- Practiced saving and uploading notebooks to GitHub

This hands-on experience provides foundational skills in version control and interactive computing.
